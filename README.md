# Abitur Checklisten 2026 – Hessen

Eine interaktive Web-App zum Abhaken aller prüfungsrelevanten Inhalte für das hessische Landesabitur 2026.

## ⚠️ Wichtiger Hinweis / Disclaimer

**Alle Angaben ohne Gewähr.**
Die in dieser App aufgeführten Inhalte basieren auf öffentlich zugänglichen Informationen und Erlassen, erheben jedoch **keinen Anspruch auf Vollständigkeit oder Richtigkeit**.
Fehler und Ungenauigkeiten können nicht ausgeschlossen werden.

**Bitte überprüfe alle Angaben stets anhand des offiziellen Abitur-Erlasses des Landes Hessen.**
Diese Checkliste ersetzt nicht die offiziellen Dokumente des Hessischen Kultusministeriums.

Offizieller Erlass: [Hessisches Kultusministerium](https://kultusministerium.hessen.de)

## 📚 Unterstützte Fächer

Aktuell sind folgende Fächer verfügbar:

- Deutsch
- Englisch
- Biologie
- Mathematik
- Politik & Wirtschaft

**Weitere Fächer folgen in Kürze!**
Die App wird kontinuierlich erweitert. Schau regelmäßig vorbei oder beobachte das Repository, um über neue Fächer informiert zu werden.

## 🚀 Nutzung

Öffne `index.html` in einem modernen Browser oder besuche die gehostete Version.

Alle Checkboxen und Selektionen werden lokal im **LocalStorage** des Browsers gespeichert. Das Löschen des Browser-Caches entfernt alle gespeicherten Einstellungen.

## 🛠️ Technologie

- HTML / CSS / JavaScript (Vanilla)
- [Tailwind CSS](https://tailwindcss.com) (via CDN)
- Fächerdaten aus `subjects.json`
